# Projected impacts of climate change to key species in the Marine Corridor of the Eastern Tropical Pacific (CMAR)

This repository supports the publication *Projected impacts of climate change to key species in the Marine Corridor of the Eastern Tropical Pacific (CMAR)* (Tentative title) submitted to peer review as part of the Lenfest Ocean Project [Fostering Climate-Resilient Biodiversity and Fisheries in Eastern Tropical Pacific MPAs](https://www.lenfestocean.org/en/research-projects/fostering-climate-resilient-biodiversity-and-fisheries-in-eastern-tropical-pacific-mpas).

# Project Goal:

 - Explore the impact of climate change on the biomass of key marine species in the Eastern Tropical Pacific
 
 - Assess regional adaptation measures aimed at reducing these impacts across ten MPAs within the CMAR

# Files and folders organization:

In this repository you will find all of the code related to the manuscript. Note that the code needs to compile data from external sources. Due to the large volume of data, these are not in GitHub. See Data Availability below.

# Data Availability 

## Primary data 

All primary data generated by this project will be found at the [UBC Dataverse Borealis Portal]() uppon publication. Metadata is also included there. See below the list of data.

- _all_years_data_abd_esm.csv_, main project dataset
- _all_cmar_mpas.shp_, shapefile with the CMAR MPAs included in this study
- _all_mpa_grid_fe.csv_, CEMAR MPAs gridded to the DBEM format

## Secondary data
All secondary data can be found online or by contacting the data owners. Please see eblow specifics.

- _SAUEEZ_July2015.shp_, This is the Sea Around Us (SAU) shapefile for the world EEZs. Contact [SAU](www.searoundus.org) for a version of it.
